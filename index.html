The Midnight Project

Data entry application targeted at Multiple Sclerosis patients

The Midnight Project is a system used to help Multiple Sclerosis patients keep track of everyday symptoms.
It consists of a questionnaire for patients to enter data, and a database that stores the patient's day to day data. This data is analyzed and notifies the patient via email of any positive or negative trends in order to figure out what makes them feel good or bad.
The questionnaire was built using QT4 and Python. The database was created with MySQL and Microsoft Excel.

Created by Kevin Huo, Toni Ogunmade, Herman Grewal and Mohamed El Shatshat for Hack 4 Health 2015. It was selected as one the top ten hacks at the event.

Real Time Space Invaders

Space Invaders clone written in C

This clone of a popular arcade game runs on a Keil Arm Cortex development board using Keil RTX. It is written in C and uses four peripherals on the board: the LCD screen that displays the game, a 4-directional joystick to handle movement, LEDs that show how many lives you've got left, and a pushbutton used to shoot.
The game runs with four threads that handle screen refresh, collision detection, score upkeep, and peripheral management.

Created by Kenneth Wong and Mohamed El Shatshat for the MTE241 course at the University of Waterloo as a cumulative project.
